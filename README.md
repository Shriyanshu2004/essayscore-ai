# EssayScore AI - NLP Writing Assessment Platform

A comprehensive peer review and essay scoring system with AI-powered feedback.

## Features

- ✅ Student enrollment and essay submission
- ✅ Automated AI scoring with detailed feedback
- ✅ Peer review system with notifications
- ✅ Browse and review other students' essays
- ✅ Real-time notifications
- ✅ Analytics dashboard
- ✅ Progress tracking

## Tech Stack

- **Backend**: FastAPI (Python)
- **Frontend**: React + Vite
- **Storage**: JSON file-based database

## Local Development

### Backend
```bash
cd backend
python main.py
```

### Frontend
```bash
cd frontend
npm install
npm run dev
```
